# Dependências

## body-parser
### Descrição
Middleware de análise de corpo Node.js.

Analise os corpos das solicitações recebidas em um middleware antes de seus manipuladores, disponíveis na req.bodypropriedade.

### Comando para instalação
```
npm i body-parser
```
